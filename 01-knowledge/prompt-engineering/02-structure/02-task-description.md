# 任务描述的 3 个层次

> 🎯 一句话核心：**任务描述越具体，AI 输出越精准**

---

## 📋 3 个关键要点

1. **层次 1：说什么** — 表面任务
2. **层次 2：为什么** — 深层目标
3. **层次 3：什么样** — 成功标准

---

## 层次 1：说什么（表面任务）

### 定义

直接告诉 AI 要做什么，最基本的任务描述。

### 示例

| 任务 | 描述 |
|------|------|
| 写作 | "写一封邮件" |
| 编程 | "写一个函数" |
| 分析 | "分析这份数据" |
| 翻译 | "翻译成英文" |

### 问题

只说"做什么"，AI 不知道：
- ❓ 为什么做（背景缺失）
- ❓ 做成什么样（标准缺失）
- ❓ 给谁用（受众缺失）

### 改进方向

→ 添加层次 2（为什么）
→ 添加层次 3（什么样）

---

## 层次 2：为什么（深层目标）

### 定义

说明任务的目的和背景，帮助 AI 理解意图。

### 示例对比

**只有层次 1：**
```
写一封邮件回复客户
```

**添加层次 2：**
```
写一封邮件回复客户。
背景：客户投诉产品延迟交货，情绪比较激动。
目标：安抚客户情绪，解释原因，给出补偿方案，
     避免客户流失。
```

### 为什么层次 2 重要？

| 维度 | 无背景 | 有背景 |
|------|--------|--------|
| 语气 | 可能不当 | 恰当（安抚） |
| 内容 | 可能遗漏 | 完整（解释 + 补偿） |
| 效果 | 可能恶化 | 改善关系 |

---

## 层次 3：什么样（成功标准）

### 定义

明确说明什么是好的输出，给 AI 清晰的评判标准。

### 示例对比

**只有层次 1+2：**
```
写一封邮件回复投诉的客户，
目的是安抚情绪、解释原因、给出补偿。
```

**添加层次 3：**
```
写一封邮件回复投诉的客户，
目的是安抚情绪、解释原因、给出补偿。

成功标准：
- 客户读完后情绪平复
- 理解延迟原因
- 接受补偿方案
- 愿意继续合作

具体要求：
- 语气诚恳但不卑微
- 解释清晰但不推卸责任
- 补偿方案具体可行
- 字数 200-300 字
```

### 成功标准类型

| 类型 | 示例 |
|------|------|
| 效果标准 | "客户愿意继续合作" |
| 格式标准 | "字数 200-300 字" |
| 质量标准 | "能直接发送" |
| 受众标准 | "非技术人员能看懂" |

## 📝 完整示例

### 示例 1：写代码

**❌ 只有层次 1：**
```
写个 Python 脚本处理 Excel
```

**✅ 三层完整：**
```
【层次 1：说什么】
写个 Python 脚本处理 Excel 文件。

【层次 2：为什么】
我每天需要处理 20+ 个销售报表，
手动合并太耗时，想自动化。
报表格式一致，需要合并成一个总表，
并计算每个产品的总销售额。

【层次 3：什么样】
成功标准：
- 能批量处理指定文件夹的所有 Excel
- 自动合并并计算
- 输出新的 Excel 文件
- 运行时间<1 分钟

具体要求：
- 使用 pandas 库
- 添加错误处理
- 包含使用示例
- 代码有注释
```

### 示例 2：写文章

**❌ 只有层次 1：**
```
写一篇 AI 科普文章
```

**✅ 三层完整：**
```
【层次 1：说什么】
写一篇 AI 科普文章。

【层次 2：为什么】
公司公众号需要吸引潜在客户，
目标读者是中小企业主，
他们对 AI 感兴趣但不了解，
需要让他们意识到 AI 能帮到业务。

【层次 3：什么样】
成功标准：
- 读者能理解 AI 基本概念
- 看到 AI 在业务中的应用场景
- 产生进一步了解的意愿
- 文章有传播性

具体要求：
- 标题吸引人（10 万 + 风格）
- 开头有故事或数据
- 中间有 3-5 个应用场景
- 结尾有行动号召
- 字数 1500-2000 字
- 配 3-5 张图建议
```

### 示例 3：分析问题

**❌ 只有层次 1：**
```
分析一下销售数据
```

**✅ 三层完整：**
```
【层次 1：说什么】
分析一下销售数据。

【层次 2：为什么】
我是销售总监，需要向 CEO 汇报 Q4 业绩。
数据包含各区域、各产品的销售额。
需要找出亮点和问题，
为明年预算和策略提供依据。

【层次 3：什么样】
成功标准：
- CEO 能快速理解业绩情况
- 亮点和问题清晰
- 建议可执行
- 能直接用于汇报 PPT

具体要求：
- 先给整体结论（1 句话）
- 再分区域、分产品分析
- 用数据支撑观点
- 给出 3 条策略建议
- 用表格展示关键数据
```

## ⚠️ 常见错误

| 错误 | 表现 | 改进 |
|------|------|------|
| 只有层次 1 | "写代码" | 添加 2+3 |
| 层次 2 空洞 | "为了学习" | 具体目标 |
| 层次 3 模糊 | "写好点" | 具体标准 |
| 层次混乱 | 混在一起说 | 分段清晰 |

## 🎯 小练习

将以下任务升级为三层描述：

**原任务：**
```
帮我做个 PPT
```

**参考答案：**
```
【层次 1：说什么】
帮我做个 PPT 大纲。

【层次 2：为什么】
我下周要给新员工做 AI 工具培训，
他们是各部门业务人员，技术背景弱，
目标是让他们了解 AI 能帮他们做什么，
培训后会安排实操练习。

【层次 3：什么样】
成功标准：
- 业务人员能理解 AI 概念
- 看到自己岗位的应用场景
- 有学习 AI 工具的动力
- 大纲能直接用于制作 PPT

具体要求：
- 10-12 页
- 每页有标题和 3-5 个要点
- 包含 2-3 个案例
- 最后有实操任务说明
- 语气轻松，不要太技术
```
