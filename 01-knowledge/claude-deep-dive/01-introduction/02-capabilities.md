# Claude 核心能力详解

> 🎯 一句话核心：**Claude 擅长理解、推理、写作，但不会实时信息**

---

## 📋 5 大核心能力

1. **文本理解** — 阅读理解、摘要、分析
2. **内容创作** — 写作、翻译、润色
3. **代码能力** — 生成、审查、调试
4. **逻辑推理** — 分析、规划、决策
5. **对话交互** — 问答、辅导、角色扮演

---

## 💡 能力详解

### 1. 文本理解 ⭐⭐⭐⭐⭐

**擅长：**
- 长文档阅读（200K 上下文）
- 核心观点提取
- 情感分析
- 主题分类

**示例：**
```
用户：请总结这篇 5000 字文章的核心观点

Claude：这篇文章的核心观点是...
主要论据有 3 个：
1. ...
2. ...
3. ...
```

**不适用：**
- 图片理解（需要视觉模型）
- 音频处理

---

### 2. 内容创作 ⭐⭐⭐⭐⭐

**擅长：**
- 商务写作（邮件、报告）
- 创意写作（故事、诗歌）
- 技术写作（文档、教程）
- 多语言翻译

**示例：**
```
用户：帮我写一封道歉邮件给客户

Claude：尊敬的 [客户姓名]：

关于 [问题]，我们深表歉意...
[详细说明原因和解决方案]

此致
敬礼
```

**不适用：**
- 需要实时数据的内容
- 个人经历类写作

---

### 3. 代码能力 ⭐⭐⭐⭐

**擅长：**
- 多种语言（Python、JS、Java 等）
- 代码生成
- 代码审查
- Bug 调试
- 代码解释

**示例：**
```
用户：用 Python 写一个快速排序

Claude：def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

**不适用：**
- 需要运行的环境
- 最新框架（知识截止前）

---

### 4. 逻辑推理 ⭐⭐⭐⭐

**擅长：**
- 问题分析
- 多步骤规划
- 利弊分析
- 决策建议

**示例：**
```
用户：我应该选择 A 工作还是 B 工作？

Claude：让我们从几个维度分析：

【薪资】A: 20k, B: 25k → B 胜
【发展】A: 大平台，B: 初创 → 看个人偏好
【地点】A: 市区，B: 郊区 → A 胜
...

综合建议：...
```

**不适用：**
- 纯数学计算（可能出错）
- 需要专业资质的建议（法律、医疗）

---

### 5. 对话交互 ⭐⭐⭐⭐⭐

**擅长：**
- 多轮对话（有记忆）
- 角色扮演
- 教学辅导
- 头脑风暴

**示例：**
```
用户：我们来角色扮演，你是面试官

Claude：好的，我是今天的面试官。
请先做一个自我介绍...
```

**不适用：**
- 情感陪伴（不是心理咨询师）
- 危险话题

## ⚠️ 能力边界

### 知道 Claude 能做什么

✅ **可以做：**
- 基于已有知识回答问题
- 创作原创内容
- 分析和总结
- 编写和解释代码
- 多轮对话

### 知道 Claude 不能做什么

❌ **不能做：**
- 访问互联网（无法查实时信息）
- 运行代码（只能生成）
- 查看图片/音频（纯文本）
- 提供专业资质建议（法律、医疗）
- 预测未来

## 🎯 小练习

判断以下任务 Claude 是否适合：

**任务 1：** 总结今天的新闻

**答案：** ❌ 不适合（Claude 无法访问实时信息）

**任务 2：** 写一封商务邮件

**答案：** ✅ 非常适合（内容创作是强项）

**任务 3：** 计算 12345 × 67890

**答案：** ⚠️ 可能出错（数学计算不是强项，建议用计算器）

**任务 4：** 分析这份 100 页的报告

**答案：** ✅ 非常适合（长文档理解是强项）
