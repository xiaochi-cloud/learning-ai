# 🤝 贡献指南

欢迎参与本项目！你的贡献会让这个仓库更有价值。

---

## 📋 如何贡献

### 1. 报告问题

发现错误、遗漏或改进建议？[提交 Issue](https://github.com/xiaochi-cloud/learning-ai/issues)

**好的 Issue 应该包含：**
- 清晰的问题描述
- 复现步骤（如适用）
- 期望结果 vs 实际结果
- 环境信息（如适用）

### 2. 提交内容

欢迎贡献以下内容：

**内容补充：**
- 学习笔记补充
- 代码示例优化
- 踩坑记录分享
- 资讯线索提供

**内容改进：**
- 错别字修正
- 表述优化
- 链接修复
- 格式统一

**功能增强：**
- 新的实践项目
- 工具/资源推荐
- 教程补充

### 3. 提交 PR

1. Fork 本仓库
2. 创建分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 开启 Pull Request

## 📝 内容规范

### 学习笔记

```markdown
# Day XXX - [主题]

## 📅 日期
YYYY-MM-DD | 第 X 周

## 🎯 今日目标
- [ ] 目标 1
- [ ] 目标 2

## 📚 学习内容
（用自己的话解释）

## 💡 关键收获
1. ...
2. ...

## 🐛 遇到的问题
- 问题 + 解决

## 🔗 参考资料
- 链接
```

### 踩坑记录

```markdown
# [错误类型] 问题简述

## 🔴 错误信息
```
完整错误日志
```

## 🔍 排查过程
1. ...
2. ...

## ✅ 解决方案
（详细步骤）

## 📌 根本原因
（深入分析）

## 🏷️ 标签
#tag1 #tag2
```

### 资讯简报

```markdown
# 📰 AI 每日快讯 | YYYY-MM-DD

## 🔥 今日头条
- **标题**：简述

## 🚀 新模型/工具
1. [产品名] 描述 + 链接

## 📚 值得读
- [标题](链接) — 一句话说明

## 🧠 点评
（简短评论）
```
