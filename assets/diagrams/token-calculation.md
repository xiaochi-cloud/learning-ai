# 📊 Token 计算详解

> 🎯 一句话核心：**Token 是 AI 处理文本的基本单位，直接影响成本和速度**

---

## 📋 什么是 Token

Token 是大语言模型处理文本的基本单位，**不是字，也不是词**。

```
英文：1 token ≈ 0.75 个单词 ≈ 4 个字符
中文：1 token ≈ 1.5 个汉字 ≈ 3 个字符
```

### 示例成本计算

**场景：** 用 Sonnet 处理一篇 1000 字文章

```
输入：1000 汉字 ≈ 650 tokens
输出：500 汉字 ≈ 325 tokens

输入成本：650 / 1,000,000 × $3 = $0.00195
输出成本：325 / 1,000,000 × $15 = $0.004875

总成本：$0.006825（约 5 分钱人民币）
```

## 🛡️ 优化 Token 使用

### 方法 1：精简 Prompt

**❌ 冗长：**
```
我希望你能够帮我分析一下这份数据，这份数据是我们公司
2024 年全年的销售数据，包含了各个区域、各个产品线、
各个月份的销售额、销售量、利润等等信息...（200 字）
```

**✅ 精简：**
```
分析 2024 年销售数据：
- 维度：区域、产品线、月份
- 指标：销售额、销售量、利润
（25 字）
```

**节省：** 87% tokens

# ❌ 设置过大（浪费）

### 方法 3：分批处理长文档

```
长文档（10000 字）
    ↓
分 5 批，每批 2000 字
    ↓
分别处理 + 汇总
    ↓
节省 30-50% tokens
```

## 📈 Token 使用监控

### Python 示例

```python
def count_tokens(text: str) -> int:
    """粗略估算 token 数"""
    chinese = sum(1 for c in text if '\u4e00' <= c <= '\u9fff')
    other = len(text) - chinese
    return int(chinese * 1.5 + other * 0.75)

# 使用
text = "你好 world"
tokens = count_tokens(text)
print(f"约 {tokens} tokens")
```
