# 🖼️ Claude API 调用流程图

---

## 完整调用流程

```
┌─────────────────────────────────────────────────────────────────┐
│                    Claude API 调用完整流程                       │
└─────────────────────────────────────────────────────────────────┘

┌──────────┐
│  用户    │
└────┬─────┘
     │ 1. 发送消息
     ↓
┌──────────────────────────────────┐
│  准备请求                         │
│  - 构建消息内容                  │
│  - 设置参数（model, max_tokens） │
│  - 附加 API Key                  │
└────────────┬─────────────────────┘
             │ 2. HTTP POST
             ↓
     ┌───────────────┐
     │  网络传输      │
     │  - DNS 解析     │
     │  - TLS 握手     │
     │  - 数据发送     │
     └───────┬───────┘
             │
             ↓
┌──────────────────────────────────┐
│  Anthropic API                   │
│  - 认证验证 (API Key)            │
│  - 速率限制检查                  │
│  - 请求解析                      │
└────────────┬─────────────────────┘
             │ 3. 处理请求
             ↓
┌──────────────────────────────────┐
│  Claude 模型                     │
│  - Token 化输入                  │
│  - 模型推理                      │
│  - 生成回复                      │
└────────────┬─────────────────────┘
             │ 4. 返回结果
             ↓
┌──────────────────────────────────┐
│  API 响应                        │
│  - 成功：200 OK + 回复内容       │
│  - 失败：错误码 + 错误信息       │
└────────────┬─────────────────────┘
             │ 5. HTTP Response
             ↓
     ┌───────────────┐
     │  网络传输      │
     └───────┬───────┘
             │
             ↓
┌──────────────────────────────────┐
│  SDK 处理响应                     │
│  - 解析 JSON                     │
│  - 提取回复内容                  │
│  - 错误处理                      │
└────────────┬─────────────────────┘
             │ 6. 返回给用户
             ↓
┌──────────┐
│  用户    │ 看到回复
└──────────┘
```

## 错误处理流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      错误处理流程                                │
└─────────────────────────────────────────────────────────────────┘

请求发送
    │
    ↓
┌─────────────┐
│  可能错误    │
└──────┬──────┘
       │
   ┌───┴───┬───────────┬──────────┬──────────┐
   │       │           │          │          │
   ↓       ↓           ↓          ↓          ↓
┌──────┐ ┌──────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ 401  │ │ 429  │ │  400   │ │  500   │ │ Timeout│
│认证  │ │速率  │ │无效请  │ │服务器  │ │ 超时   │
│失败  │ │限制  │ │求      │ │错误    │ │        │
└──┬───┘ └──┬───┘ └───┬────┘ └───┬────┘ └───┬────┘
   │        │        │          │          │
   │        │        │          │          │
   ↓        ↓        ↓          ↓          ↓
检查    等待    修改      重试      重试
API     后      请求      或        或
Key     重试    参数      联系      检查
                支持      网络
```
