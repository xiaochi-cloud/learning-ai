# [Prompt] 输出格式不符合预期怎么办

> 🎯 一句话解决：**明确格式要求 + 提供示例 + 约束强化**

---

## 🔴 问题表现

```
你要求：用表格展示
AI 输出：bullet points

你要求：JSON 格式
AI 输出：自然语言

你要求：分 3 点
AI 输出：一大段文字
```

---

## ✅ 解决方案

### 方案 1：明确格式描述

**❌ 模糊要求：**
```
整理一下这些数据
```

**✅ 明确要求：**
```
用以下格式整理数据：

| 区域 | 销售额 | 增长率 | 排名 |
|------|--------|--------|------|
| 华东 | 100 万  | 15%    | 1    |
| 华北 | 80 万   | 12%    | 2    |

要求：
- 表格形式
- 包含 4 列（如上）
- 按销售额降序排列
```

---

### 方案 2：提供格式示例

**❌ 无示例：**
```
输出 JSON 格式
```

**✅ 有示例：**
```
输出 JSON 格式，结构如下：

```json
{
  "summary": "一句话总结",
  "points": [
    {"title": "要点 1 标题", "content": "要点 1 内容"},
    {"title": "要点 2 标题", "content": "要点 2 内容"}
  ],
  "conclusion": "结论"
}
```

请严格按照以上格式输出。
```

---

### 方案 3：约束强化

**技巧：在开头和结尾都强调格式**

```
【重要】请严格按照以下格式输出：

[格式要求]
- 用表格
- 3 列：姓名、年龄、城市
- 按姓名排序

[任务内容]
整理以下人员信息...

【再次强调】输出格式：
- 必须是表格
- 必须是 3 列
- 必须按姓名排序
不要添加任何额外内容。
```

## 📝 常见格式要求模板

### 表格格式

```
请用表格形式展示，包含以下列：
| 列 1 | 列 2 | 列 3 |

要求：
- 表头清晰
- 数据对齐
- 按 XX 排序
```

### JSON 格式

```
请输出 JSON 格式，结构如下：

```json
{
  "key1": "value1",
  "key2": ["item1", "item2"],
  "key3": {
    "nested": "value"
  }
}
```

注意：
- 不要添加 Markdown 标记
- 确保 JSON 有效
- 用双引号
```

### Bullet Points

```
请用 bullet points 列出，格式如下：

• 要点 1：标题 + 详细说明（1-2 句）
• 要点 2：标题 + 详细说明（1-2 句）
• 要点 3：标题 + 详细说明（1-2 句）

要求：
- 每个要点以•开头
- 先标题后说明
- 说明不超过 2 句
```

### 分步骤

```
请分步骤说明，格式如下：

**步骤 1：** [步骤名称]
[详细说明]

**步骤 2：** [步骤名称]
[详细说明]

**步骤 3：** [步骤名称]
[详细说明]

要求：
- 步骤用粗体标记
- 每步有名称和说明
- 说明包含具体操作
```

## ⚠️ 常见错误

| 错误 | 表现 | 改进 |
|------|------|------|
| 格式模糊 | "整理一下" | 具体格式描述 |
| 只有文字 | "用表格"但无示例 | 提供表格示例 |
| 指令冲突 | "简洁但要详细" | 统一指令 |
| 位置隐蔽 | 格式要求在中间 | 放在开头 + 结尾 |
