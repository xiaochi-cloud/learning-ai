## 🚀 运行指南

### 1. 安装依赖

```bash
pip install anthropic
```

### 2. 设置 API Key

```bash
# macOS/Linux
export ANTHROPIC_API_KEY='your-api-key-here'

# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="your-api-key-here"
```

### 3. 运行脚本

```bash
python hello_claude.py
```

### 4. 预期输出

