## 📚 学习内容

### 1. Claude 是什么

Claude 是 Anthropic 公司开发的大语言模型（LLM），特点：

- **定位：** AI 助手，帮助完成写作、分析、编程等任务
- **优势：** 逻辑清晰、长上下文、安全性高
- **版本：** Claude 3 系列（Haiku/Sonnet/Opus）

> 💡 **我的理解：** Claude 就像一个博学但谨慎的助手，你给它清晰的任务，它给你高质量的输出。

### 2. 第一次对话体验

**我的第一个 prompt：**
```
你好，我想学习 AI 应用开发，应该从哪里开始？
```

**Claude 的回答要点：**
1. 先了解基础概念（LLM、prompt、API）
2. 选择一门编程语言（推荐 Python）
3. 动手做小项目
4. 持续学习跟进

> 💡 **收获：** 好的问题得到结构化回答，prompt 质量直接影响输出质量。

### 3. 提示词基础

**什么是提示词（Prompt）？**
- 你给 AI 的指令
- 决定了 AI 如何理解和回应

**好提示词的要素：**
```
1. 清晰的任务描述
2. 必要的背景信息
3. 期望的输出格式
4. 约束条件（如有）
```

**示例对比：**

❌ 差：`写个 Python 代码`

✅ 好：
```
用 Python 写一个函数，功能是：
- 输入：一个文件路径
- 输出：文件的行数统计
- 要求：处理异常，返回整数
```

---

## 💡 关键收获

1. **AI 不是魔法** — 输出质量取决于输入质量
2. **清晰比聪明重要** — 简单的清晰指令胜过复杂的模糊指令
3. **迭代是常态** — 第一次不完美很正常，继续优化
4. **动手最重要** — 看再多教程不如自己试一次

---

## 🐛 遇到的问题

### 问题 1：不知道问什么

**现象：** 面对 AI 大脑一片空白

**解决：**
- 从实际问题出发（"帮我写个 XX 脚本"）
- 参考官方示例
- 看别人怎么用（社区、教程）

### 问题 2：回答太长看不完

**现象：** Claude 输出几千字，抓不住重点

**解决：**
```
请在 200 字以内回答，用 bullet points 列出关键点
```
