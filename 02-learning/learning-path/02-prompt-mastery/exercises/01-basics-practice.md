## 练习 1：优化差 Prompt

### 任务

以下 Prompt 有什么问题？如何优化？

```
写个代码
```

### 要求

应用"提示词 4 要素"：
1. 任务清晰
2. 背景充分
3. 格式具体
4. 约束明确

### 参考答案

```
【任务】
用 Python 写一个函数，统计文本中每个单词的出现频率。

【背景】
我是编程初学者，正在学习 Python 字典和字符串处理。
这个函数将用于我的第一个小项目：文本分析工具。

【格式要求】
- 函数名：word_frequency
- 输入：一个字符串（文本）
- 输出：一个字典（单词→频率）
- 包含详细的注释
- 提供 3 个使用示例

【约束条件】
- 代码简洁，不超过 30 行
- 使用 Python 内置功能，不要第三方库
- 忽略大小写差异
- 去除标点符号
```

## 练习 2：角色设定

### 任务

为以下任务设定合适的角色：

```
任务：帮我分析这份财务报表
```

### 要求

角色设定包含：
- 身份
- 专业领域
- 经验水平
- 风格特点

### 参考答案

```
你是一位 CFA 持证的财务分析师，
有 8 年投行经验，
服务过 30+ 家上市公司，
擅长发现财务数据背后的问题和风险，
分析风格数据驱动，结论明确，
会用通俗语言解释专业概念。
```

## 练习 3：任务描述 3 层次

### 任务

将以下任务升级为三层描述：

```
分析一下销售数据
```

### 要求

- 层次 1：说什么（表面任务）
- 层次 2：为什么（深层目标）
- 层次 3：什么样（成功标准）

### 参考答案

```
【层次 1：说什么】
分析一下销售数据。

【层次 2：为什么】
我是销售总监，需要向 CEO 汇报 Q4 业绩。
数据包含各区域、各产品的销售额。
需要找出亮点和问题，为明年预算提供依据。

【层次 3：什么样】
成功标准：
- CEO 能快速理解业绩情况
- 亮点和问题清晰
- 建议可执行
- 能直接用于汇报 PPT

具体要求：
- 先给整体结论（1 句话）
- 再分区域、分产品分析
- 用数据支撑观点
- 给出 3 条策略建议
- 用表格展示关键数据
```

## 练习 4：格式控制

### 任务

要求 AI 输出 JSON 格式，如何确保格式正确？

### 要求

应用"格式控制 5 种方法"中的至少 2 种。

### 参考答案

```
请输出 JSON 格式，结构如下：

```json
{
  "summary": "一句话总结",
  "points": [
    {"title": "要点 1 标题", "content": "要点 1 内容"},
    {"title": "要点 2 标题", "content": "要点 2 内容"}
  ],
  "conclusion": "结论"
}
```

【重要】请严格按照以上格式输出：
- 只输出纯 JSON
- 不要 Markdown 标记
- 不要添加任何解释
- 确保 JSON 有效（双引号、逗号等）
```

**使用方法：**
- 方法 2：提供示例
- 方法 5：约束强化

## 练习 5：综合应用

### 任务

写一个完整的 Prompt，包含：
- 角色设定
- 任务描述（3 层次）
- 格式要求
- 约束条件

### 场景

你需要 AI 帮你写一封商务邮件，回复客户的投诉。

### 参考模板

```
【角色】
你是一位有 10 年经验的客服经理，
擅长处理客户投诉和危机沟通，
语气专业但友善，能平衡公司利益和客户满意度。

【任务】
写一封邮件回复投诉的客户。

【背景】
客户投诉产品延迟交货，情绪比较激动。
我们延迟的原因是供应链问题，预计还要 1 周才能发货。
我们希望安抚客户情绪，解释原因，给出补偿方案，
避免客户流失和负面评价。

【格式要求】
请按以下结构：
1. 称呼和感谢
2. 诚恳道歉
3. 解释原因（不推卸责任）
4. 补偿方案（具体）
5. 下一步行动
6. 结束语

【约束条件】
- 语气诚恳但不卑微
- 字数 200-300 字
- 补偿方案：9 折优惠 + 免费加急
- 不要承诺做不到的事
- 用中文
```
