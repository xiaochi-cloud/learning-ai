---

## 1️⃣ 这是什么？

这是一个**真实学习记录**仓库，记录从 0 开始学习 AI 应用开发的全过程。

**核心特点：**
- ✅ 一手学习笔记 — 不是文档搬运
- ✅ 可运行代码 — 每段代码都经过测试
- ✅ 完整踩坑记录 — 错误 + 排查 + 解决
- ✅ 每日更新 — 持续 12 周

---

## 2️⃣ 适合谁？

**适合：**
- 想学 AI 但不知道从哪开始
- 有编程基础，想学 AI 应用开发
- 喜欢边学边做，不只是看理论

**不适合：**
- 想找 7 天速成教程
- 只想抄代码不想思考
- 期待完美答案（学习是迭代过程）

## 4️⃣ 怎么学？

### 推荐学习流程

```
1. 看每日笔记 → 了解今天学什么
2. 读学习内容 → 理解核心概念
3. 运行代码 → 动手实践
4. 记录问题 → 形成自己的笔记
5. 持续迭代 → 每天进步一点
```

### 学习时间建议

| 投入 | 建议 |
|------|------|
| 每天 30 分钟 | 看笔记 + 读资讯 |
| 每天 1 小时 | + 运行代码示例 |
| 每天 2 小时 | + 动手实践 + 记录 |

## 5️⃣ 常见问题

### Q: 需要编程基础吗？

A: 需要基础编程知识（变量、函数、循环），Python 最佳。不会的话先学 Python 基础。

### Q: 需要付费吗？

A: 学习免费。使用 Claude API 需要付费，但有免费额度。本地模型完全免费。

### Q: 跟得上更新吗？

A: 内容持续更新，建议从最新内容开始，或按路线图系统学习。

### Q: 可以提问吗？

A: 欢迎提 Issue！看到就会回复。
