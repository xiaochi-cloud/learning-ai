# 📖 术语表

AI 学习常用术语解释，按字母排序。

---

## A

### Agent (智能体)
能自主完成复杂任务的 AI 系统，通常包含规划、工具使用、记忆等能力。

**示例：** 一个能自动研究课题、写报告、发邮件的 AI 助手。

### API (应用程序接口)
不同软件之间通信的规范。AI API 让你能调用大模型的能力。

**示例：** Claude API 让你用代码调用 Claude。

### Anthropic
AI 安全公司，Claude 的开发者。

---

## C

### Claude
Anthropic 开发的大语言模型系列，以安全和逻辑清晰著称。

**版本：** Haiku（快）、Sonnet（平衡）、Opus（强）

### Context (上下文)
AI 能记住的对话或文本长度。上下文越长，能处理的内容越多。

**示例：** 100K context 能处理约 75000 字的文档。

### Chain of Thought (思维链)
让 AI 逐步推理的提示技巧，提升复杂任务准确率。

**示例：** "请逐步思考：第一步...第二步..."

---

## E

### Embedding (嵌入)
将文本转换为数值向量的技术，用于语义搜索、聚类等。

**示例：** 相似意思的句子有相近的 embedding。

---

## L

### LLM (大语言模型)
Large Language Model，基于海量文本训练的语言模型。

**代表：** GPT、Claude、Llama、通义千问

### LangChain
流行的 AI 应用开发框架，简化 LLM 集成。

**功能：** Chain、Agent、Memory、Tool 等

### Local LLM (本地大模型)
在本地电脑运行的开源大模型，无需联网。

**工具：** Ollama、LM Studio

---

## P

### Prompt (提示词)
给 AI 的指令，决定 AI 如何回应。

**好 Prompt：** 清晰、具体、有约束

### Prompt Engineering (提示词工程)
设计和优化 prompt 的技巧和方法。

**目标：** 让 AI 输出更准确、更符合预期

### Python
编程语言，AI 开发首选。

**特点：** 语法简洁、生态丰富

---

## R

### RAG (检索增强生成)
Retrieval-Augmented Generation，先检索相关知识再生成回答。

**用途：** 知识库问答、文档查询

**优势：** 减少幻觉，提供准确信息

### Role (角色)
在 prompt 中给 AI 设定的身份，影响回答风格。

**示例：** "你是一位资深 Python 工程师..."

---

## T

### Token
AI 处理文本的基本单位。英文约 1 token = 0.75 单词，中文约 1 token = 1.5 字。

**计费：** API 通常按 token 收费

### Temperature (温度)
控制 AI 输出随机性的参数。

- 低温（0.2）：确定、保守
- 高温（0.8）：创意、多样

### Tool (工具)
AI 可以调用的外部功能，如搜索、计算、API 调用等。

**示例：** 让 AI 调用天气 API 查询天气。

### Transformer
现代大模型的基础架构。

**特点：** 自注意力机制，擅长处理序列数据

---

## 其他

### 🤗 Hugging Face
AI 社区和平台，提供模型、数据集、工具。

### 📚 Few-shot Learning
给 AI 几个示例让它学习模式。

**示例：** 
```
输入："你好" → 输出："你好！有什么可以帮你？"
输入："再见" → 输出："再见！欢迎下次再来。"
输入："谢谢" → 输出："？"
```

### 🎯 Zero-shot Learning
不给示例，直接让 AI 完成任务。

### 🐛 Hallucination (幻觉)
AI 编造不存在的信息。

**应对：** RAG、要求引用来源、验证关键信息
