常见问题解答，持续更新。

## 📚 学习相关

### Q: 我完全零基础，能学吗？

**A:** 建议先学 Python 基础（变量、函数、循环、文件操作），大概需要 2-4 周。有基础后从这里开始正合适。

**推荐资源：**
- [Python 官方教程](https://docs.python.org/3/tutorial/)
- [廖雪峰 Python 教程](https://www.liaoxuefeng.com/wiki/1016959663602400)

---

### Q: 学完这个能找到工作吗？

**A:** 这个仓库聚焦 AI 应用开发技能，是求职的加分项，但不是全部。还需要：
- 扎实的编程基础
- 项目经验（可以用仓库里的项目练手）
- 系统设计能力
- 持续学习能力

---

### Q: 12 周能学完吗？

**A:** "学完"是相对的。12 周后你会：
- ✅ 掌握 AI 应用开发核心技能
- ✅ 能独立开发 AI 应用
- ✅ 建立持续学习的基础

但 AI 领域变化快，学习是持续过程。

## 💻 技术相关

### Q: 必须用 Python 吗？

**A:** 推荐 Python，因为：
- AI 生态最丰富
- 学习资源最多
- 语法简洁易学

但其他语言也可以（JavaScript、Go 等），原理相通。

---

### Q: 必须用 Claude 吗？

**A:** 不必须。仓库以 Claude 为主，但：
- 概念和技巧通用
- 代码稍改就能用于其他模型
- 后续会有多模型对比

---

### Q: API 很贵吗？

**A:** Claude API 价格（参考）：
- Haiku: $0.25 / 百万 tokens（输入）
- Sonnet: $3.00 / 百万 tokens（输入）
- Opus: $15.00 / 百万 tokens（输入）

**学习成本：** 每天$1-2 足够。

**省钱技巧：**
- 用 Haiku 做简单任务
- 本地模型免费（Ollama）
- 利用免费额度

---

### Q: 国内能用吗？

**A:** 
- Claude 官网：需要特殊网络
- Claude API：需要特殊网络
- 替代方案：通义千问、智谱 GLM 等国产模型

仓库内容对国产模型同样适用。

## 🛠️ 使用相关

### Q: 代码运行报错怎么办？

**A:** 
1. 检查 Python 版本（推荐 3.9+）
2. 检查依赖是否安装（`pip install -r requirements.txt`）
3. 检查环境变量（API Key 等）
4. 查看踩坑板块是否有类似问题
5. 提 Issue 描述详细错误信息
