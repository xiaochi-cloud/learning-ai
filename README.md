## 📖 这是什么

这是一个**真实学习记录**仓库，记录从 0 开始系统学习 AI 应用开发的完整过程。

### 核心价值

```
✅ 一手学习记录 — 不是文档搬运，是真实学习过程
✅ 可运行代码 — 每段代码都经过测试
✅ 完整踩坑记录 — 错误 + 排查 + 解决，节省你试错时间
✅ 每日资讯精选 — 跟上 AI 领域最新动态
✅ 可持续更新 — 12 周系统规划，不是虎头蛇尾
```

### 不适合谁

```
❌ 想找速成教程的
❌ 只想抄代码不想思考的
❌ 期待完美答案的（学习是迭代过程）
```

## 🗺️ 12 周学习路线

| 周次 | 主题 | 核心产出 |
|------|------|----------|
| W01 | Claude 入门 | 基础概念 + 第一个 Agent |
| W02 | 提示词工程 | 技巧库 + 案例集 |
| W03 | API 集成 | 钉钉/微信机器人 |
| W04 | 自动化脚本 | 文件/邮件/数据处理 |
| W05 | RAG 基础 | 知识库问答系统 |
| W06 | Agent 框架 | LangChain/LLamaIndex |
| W07 | 多模型对比 | Claude vs GPT vs Qwen |
| W08 | 部署实践 | 本地部署 + 云服务 |

| W10 | 安全合规 | 数据隐私 + 使用规范 |
| W11 | 综合项目 | 完整应用开发 |
| W12 | 复盘迭代 | 内容优化 + 路线图 2.0 |
